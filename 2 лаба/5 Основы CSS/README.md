# Задание 5 — Основы CSS

## Описание
В данном задании была выполнена стилизация ранее созданной HTML-страницы с формами.

## Реализовано
- подключение внешнего CSS-файла
- оформление фона страницы и текста
- стилизация заголовков и ссылок
- добавление отступов между секциями
- рамки у блоков
- использование селекторов:
  - по тегу
  - по классу
  - по id
  - вложенных селекторов
  - групповых селекторов
  - псевдокласса :hover

Сложностей при выполнении задания не возникло
