# Задача 1 — Первая HTML-страница

## Описание
В рамках задания была создана первая HTML-страница с базовой структурой и основными элементами HTML.

## Реализовано
- Заголовок страницы (title)
- Главный заголовок (h1)
- 3 абзаца текста
- Список из 5 пунктов
- Ссылка на внешний сайт
- Кнопка
- Горизонтальная линия
- Подвал страницы
- Дополнительные блоки

## Сложности
Сложностей при выполнении задания не возникло
