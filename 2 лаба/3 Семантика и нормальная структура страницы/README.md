# Задача 3 — Страница «Новости»

## Описание
Создана HTML-страница новостей с использованием семантической разметки HTML5.

## Реализовано
- Шапка сайта (header)
- Основной блок (main)
- Блок новостей (section)
- 6 новостных статей (article)

Каждая новость содержит:
- заголовок
- дату публикации
- текст
- ссылку «Читать дальше»

- Подвал страницы (footer)

Сложностей при выполнении задания не возникло
